# 🚀 Smart File Organizer

A simple Python automation project that automatically organizes files into folders based on their file types.

## 📌 Overview

Managing folders with mixed file types can become messy and time-consuming. This project helps automate that process by sorting files into categorized folders such as Images, Documents, Videos, Music, Archives, and Python Files.

The script scans a selected directory, creates folders if they do not already exist, and moves files into their appropriate locations automatically.

---

## ✨ Features

* Automatically organizes files by extension
* Creates folders if they don't exist
* Supports multiple file categories
* Beginner-friendly and easy-to-understand code
* Helps keep directories clean and organized

---

## 🛠 Technologies Used

* Python
* os Module
* shutil Module

---

## 📂 Supported Categories

| Category     | Extensions               |
| ------------ | ------------------------ |
| Images       | .jpg, .jpeg, .png, .gif  |
| Documents    | .pdf, .docx, .txt, .pptx |
| Videos       | .mp4, .mkv, .mov         |
| Music        | .mp3, .wav               |
| Archives     | .zip, .rar               |
| Python Files | .py                      |

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-File-Organizer.git
```

### 2. Navigate to the Project Directory

```bash
cd Smart-File-Organizer
```

### 3. Run the Script

```bash
python organizer.py
```

### 4. Enter Folder Path

```text
Enter folder path:
```

Provide the path of the folder you want to organize.
